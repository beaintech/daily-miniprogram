# daily-miniprogram

每天早上 6:00，AI 团队自动开会讨论并开发一个新的微信小程序。

## 项目说明

本项目由 AI Agent 团队（PM / Developer / Designer / Tester）自动协作开发：
- **触发时间**：每天早上 6:00（北京时间）
- **开发流程**：团队讨论 → 需求设计 → 代码编写 → 测试验证 → GitHub 推送
- **技术栈**：微信小程序原生开发

## 已完成的小程序

### #1 - 解压乐园（戳泡泡纸）✅
- 满屏彩色泡泡，手指点按/滑动戳破
- 7种颜色随机排列，12x6 泡泡网格
- 粒子爆破特效（8个粒子向四周扩散）
- 手机震动反馈（连续戳时增强震动）
- 滑动连续戳泡泡（手指划过自动戳破）
- 本地记录最高分
- 完成后弹出庆祝弹窗 + 长震动

### #2 - 超级按钮（即将上线）
- 各种开关、按钮、滑块
- 纯粹满足强迫症

### #3 - 切沙子（即将上线）
- 手指划过沙堆，看沙子落下

## 如何运行

1. 下载 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 导入本项目
3. 使用测试 AppID 或自己的 AppID
4. 点击预览即可
